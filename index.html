Features

Pbinfo tools

With this tool, you no longer need a browser to submit your problem code. With just a click, you can submit your code on pbinfo directly from this app.

Fetch test cases

You can fetch test cases from different competitive platforms with just a click. Now you no longer need to copy and paste samples.

Clean interface

Code Nimble design is user-friendly, super fast, and very customizable. You will feel great using this application.

Templates

Automate your work with this feature! This will create a new custom file with a custom code.

Run & Support

Run your C++ & Python files with just a click in just a matter of time.

Paint window

With this tool, you no longer need to open paint app to do your sketches and notes. You can draw whatever you want.

Local server

You want to cooperate with your friend on the same problem or you have a team competition? This will help you and your team edit the code live for everyone.

Input, output & expected output

Automate your code reviewing with these inputs. Run with the pre-input and compare output and expected output to see if your code will get the full score.
