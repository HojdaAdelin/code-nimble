Code Nimble - Version 2.0

We’re thrilled to announce the latest update to Code Nimble, now at version 2.0. This release introduces several exciting features, improvements, and fixes to enhance your coding experience. Here’s a detailed overview:

New Features


Python Support

Enhanced support for Python, including improved syntax highlighting and execution capabilities.

Scrollbar in Suggestions

A scrollbar has been added to the suggestions box for easier navigation through autocomplete options.

Right Panel

A new right panel interface for better organization and access to tools and information.

New Input & Output System

Revamped input and output handling system for clearer interaction and debugging.

Expected Output and Check Test Cases

Ability to define expected output and check test cases directly within the application.

Submit Code to pbinfo.ro

Direct submission of code to pbinfo.ro from within the Code Nimble app.

Open Site Function in Home

Quick access feature to open relevant websites directly from the home interface.

New Code Highlight System

Improved code highlighting system for better readability and visual distinction.

New Submit Code Button

Streamlined process with a new button for submitting code to various platforms.

Fetch Test Cases Support

Added support for fetching test cases from multiple online judges including pbinfo.ro, kilonova.ro, codeforces.com, and atcoder.jp.

New Change Log in App

Integrated changelog within the app for tracking updates and changes.

Run Python Files

Ability to execute Python files directly within the application environment.

New Local Server Interface

Enhanced local server interface for real-time collaboration and code sharing.

Added Buttons for Theme System

Additional buttons for easier management and customization of visual themes.

Clear Terminal Function

New function to clear the terminal output for better visibility.

Code Formatting

Improved code formatting options to maintain consistency and readability.

Output Comparator

Added functionality to compare output results for better debugging.

Highlight Color Palette in Themes JSON

Extended color palette options in the themes JSON file for more customization.

Fixed/Changes


Fixed Delete File Function

Resolved issues with file deletion functionality.

Changed Template & Theme Interface

Updated interface for easier template and theme management.

Fixed Suggestion Box

Issues with suggestion box behavior have been fixed for smoother autocomplete.

Changed Open Recent List

Improved handling and display of recently opened files.

Fixed Exit Function

Resolved issues related to application exit behavior.

Changed Treeview Indicator

Enhanced visual indicators in the treeview for better navigation.

Removed Guide from Home

Removed redundant guide feature from the home interface.

Fixed Run Function

Fixed issues with the run function to ensure consistent behavior.

Fixed Tab Function for Suggestions

Resolved issues related to tab behavior in suggestion boxes.

Fixed Templates

Template functionality has been fixed and improved.

Fixed Double Paste Issue

Resolved issues with unintended double paste actions.

UTF-8 Support

Added support for UTF-8 encoding for broader language compatibility.

We appreciate your continued support and feedback. This update reflects our commitment to providing you with the best tools for coding and programming. Enjoy the new features and enhancements!
