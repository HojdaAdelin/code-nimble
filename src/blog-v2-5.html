Code Nimble - Version 2.5.0

We are excited to introduce the latest update to Code Nimble, now at version 2.5.0. This release brings a host of new features, enhancements, and fixes to optimize your coding experience. Below is a detailed overview:

New Features


Kilonova.ro Submit Function

Submit your code directly to Kilonova.ro from within the application, offering a seamless and efficient submission experience.

New Light-Fire Theme

A fresh, dynamic theme that brings a vibrant and energetic visual style to your coding environment.

"Submit Code" Menu in Right Panel

A new dedicated menu in the right panel centralizes submission functions, streamlining access to various platforms.

Extended Testing Tab Functionality

The Testing tab now supports up to 5 test sets, allowing users to run multiple comprehensive tests for their code within the app.

Settings Tab in Right Panel

A new tab in the right panel provides easy access to customize and manage application settings.

Pre-Template on Startup

Automatically loads a default template when you start the application, helping you kickstart your coding quickly.

Competitive Companion Support

Integration with Competitive Companion ensures smooth problem parsing for competitive programming tasks.

Fixed/Changes


New Highlighter Colors

Updated highlighter colors for Light, Dark, Ocean, and Dark-Blue themes, improving syntax clarity and readability.

Fixed Autocompletion Templates

Resolved issues with autocompletion templates in the editor to enhance coding efficiency and accuracy.

Fixed Documentation Bug

Addressed a bug in the documentation viewer to ensure a smoother and more reliable user experience.

Thank you for your continued support and feedback. This update reflects our commitment to providing you with the best tools for coding and programming. Enjoy the new features and enhancements!
