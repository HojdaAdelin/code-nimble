Code Nimble - Version 2.6.0

We are excited to introduce the latest update to Code Nimble, now at version 2.6.0. This release brings a host of new features, enhancements, and fixes to optimize your coding experience. Below is a detailed overview:

New Features

Encryption for username & password in app_data_

For extra security, we've implemented encryption for the username and password in app_data_. This ensures that your credentials are protected from unauthorized access.

Fixed/Changes

Improved github repo structure

We've streamlined the github repo structure, making it easier to navigate and access the codebase.

Fixed autocompleter insertion

We've fixed a bug that caused autocompleter suggestions to be inserted incorrectly in the editor.


Thank you for your continued support and feedback. This update reflects our commitment to providing you with the best tools for coding and programming. Enjoy the new features and enhancements!
