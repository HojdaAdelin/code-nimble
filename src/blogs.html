Code Nimble - Version: 2.0

We’re thrilled to announce the release of Code Nimble 2.0! This major update brings a wealth of new features, enhancements, and fixes designed to enhance your programming experience. Here’s a detailed look at what’s new and what has been improved:

New Features

Server Status in Status Bar

Monitor the status of your local server directly from the status bar, providing real-time updates on server activity and connectivity.

Auto Save

Files will now automatically save every 5 seconds, ensuring that your work is preserved with minimal manual intervention.

Ctrl+Roller Bind

Enhance your navigation and editing experience with a new bind for Ctrl+Roller, allowing for smoother scrolling and zooming.

Refresh Editor

Use Ctrl+Shift+R to quickly refresh the editor, updating your workspace and clearing any temporary glitches.

Save Session

Save your current session, including open files and editor state, so you can easily resume work later.

Summer Theme

Enjoy a fresh, new Summer theme that adds a vibrant touch to your coding environment.

Drag and Drop Files

Drag and drop files directly into the editor for quicker and easier file management.

Exit Accelerator

Accelerate the process of exiting the application with a new exit accelerator feature.

Open Recent

Quickly access recently opened files through the new Open Recent functionality.

LAN Password

Secure your local server with a LAN password to restrict access and protect your code.

Terminal Inside Editor

Use an integrated terminal within the editor for seamless command-line access and execution.

Theme Changer

Switch between themes effortlessly with the new theme changer feature.

Tags in Suggestions

View and use tags in code suggestions to enhance code completion and context-awareness.

Snippet Code Interface

Manage and insert code snippets more effectively with the new snippet code interface.

Snippets in Suggestions

Code snippets are now integrated into suggestions for faster and more efficient coding.

Kilonova.ro Tools

Integration with Kilnova.ro tools adds additional functionality and support for competitive programming.

Python Support

Extended support for Python allows you to write, run, and manage Python code seamlessly within Code Nimble.

Scrollbar in Suggestions

Navigate through code suggestions more easily with the addition of a scrollbar.

Right Panel

A new right panel provides additional space for tools, views, and customizations.

New Input & Output System

Revamped system for handling input and output, including expected output and test case checks.

Submit Code Directly to Pbinfo.ro

Submit your code directly to Pbinfo.ro from within the app for streamlined competitive programming.

Open Site Function in Home

Easily open websites from the Home screen for quick access to resources and documentation.

New Code Highlight System

Experience improved code highlighting with a new system for better readability and syntax awareness.

New Submit Code Button

A new, more accessible submit button makes it easier to submit your code for evaluation.

Fetch Test Cases Support

Fetch test cases from multiple platforms, including Pbinfo.ro, Kilnova.ro, Codeforces.com, and Atcoder.jp.

New Change Log in App

View the change log directly within the app to keep track of updates and new features.

Run Python Files

Execute Python files directly within the editor, enhancing the versatility of your coding environment.

New Local Server Interface

A revamped local server interface improves usability and management of collaborative coding sessions.

Theme System Buttons

New buttons for the theme system provide easy access to theme management and customization.

Clear Terminal Function

Clear the terminal with a single function to maintain a clean and organized workspace.

Code Formatting

Improved code formatting features enhance readability and maintain coding standards.

Output Comparator

Compare outputs directly within the editor to verify code correctness and debug issues efficiently.

Highlight Color Palette

Add custom highlight colors to themes using the new color palette feature in the JSON theme files.

Paint Mode Enhancements

In Paint Mode, clicking on a color now changes the drawing tool to a pencil for more precise editing.

Go to Line Feature

Quickly navigate to specific lines in your code with the new Go to Line feature.

Fixes and Changes

Fixed Delete File Function

Resolved issues with the delete file function to ensure proper file management.

Changed Template & Theme Interface

Updated the interface for templates and themes for a more streamlined user experience.

Fixed Suggestion Box

Addressed problems with the suggestion box to improve code completion accuracy.

Changed Open Recent List

Updated the Open Recent list to better reflect recent file usage and access.

Fixed Exit Function

Resolved issues with the application exit function to ensure a smooth closing experience.

Changed Treeview Indicator

Improved the treeview indicator for better visual feedback and navigation.

Removed Guide from Home

The Guide has been removed from the Home screen to streamline the user interface.

Fixed Run Function

Addressed issues with the run function to ensure reliable code execution.

Fixed Tab Function for Suggestions

Corrected problems with tab navigation in code suggestions for improved usability.

Fixed Templates

Resolved issues with templates to ensure they function correctly and efficiently.

Fixed Double Paste Issue

Addressed the problem of double pasting to prevent duplicate content errors.

UTF-8 Support

Added support for UTF-8 encoding to handle a wider range of characters and symbols.

Thank you for your continued support and feedback. We’re committed to making Code Nimble the best tool for competitive programming. Enjoy the new features and improvements in version 2.0!

Code Nimble - Version: 1.5

We’re excited to announce the latest update to Code Nimble, now at version 1.5. This release brings a host of new features, enhancements, and fixes to improve your programming experience. Here’s a breakdown of what’s new and what’s been fixed:

New Features

Paint Mode

We’ve integrated a paint window directly into the app, allowing you to draw and make notes without switching to an external paint application. You can now easily change colors within this mode and use tabs for better organization.

Local Server

This new feature enables real-time collaboration with others. You can now work on code simultaneously with your teammates using the local server functionality, which includes a connected users list in the server panel.

Profile Management

Manage your profiles directly within Code Nimble, making it easier to switch between different coding setups or user preferences.

Autocomplete Enhancements

C++ autocompletion is now more intuitive. Pressing ENTER will automatically complete the code when suggestions are mapped. We’ve also added autocomplete support for symbols like ", ', and *.

Improved File Handling

A new button bind in the file tab enhances your navigation experience, and a "*" indicator in the file tab shows when a file is modified, so you never lose track of unsaved changes.

Settings and Customization

The new theme system, using JSON files, allows you to easily switch between different visual themes, including the new Ocean theme. Customization options now include settings for the status bar and text highlighting for Python.

Enhanced File Execution

You can now run the content of the textbox directly without saving the file each time, streamlining your workflow. The file tab pack issue has also been resolved to prevent loss of tabs when none are open.

New Documentation

We’ve added CHANGELOG.md and CONTRIBUTION.md files for better tracking of updates and to guide contributors.

Fixes

Autocompletion Corrections

We’ve fixed issues with autocompletion when the suggestion list was not mapped correctly.

Editor Stability

Issues with Ctrl+Backspace and file saving have been addressed to ensure smoother operation.

UI Enhancements

We resolved issues with Input & Output display in the treeview and improved the redraw function after find & replace operations.

Text Highlighting

Text highlighting in the editor has been updated for better visibility and accuracy.

Cursor Positioning

The cursor position now updates correctly when using ENTER between curly braces, enhancing code formatting and readability.

Thank you for your continued support and feedback. We’re committed to making Code Nimble the best tool for competitive programming, and this update is a testament to our dedication. Enjoy the new features and improvements!
